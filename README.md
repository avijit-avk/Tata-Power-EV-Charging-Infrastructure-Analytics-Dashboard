# ⚡ EV Charging Infrastructure Analytics Dashboard

An end-to-end **Data Analytics** project built in **Microsoft Excel** to evaluate the operational performance, customer behavior, station utilization, and financial feasibility of EV charging infrastructure.

The project transforms raw operational data into interactive executive dashboards that enable data-driven business decisions for infrastructure planning, investment prioritization, and operational optimization.

---

## 📌 Project Overview

With the rapid adoption of electric vehicles, charging infrastructure operators require actionable insights to optimize station performance and maximize return on investment.

This project analyzes charging station operations using Excel's analytics capabilities to answer critical business questions related to utilization, customer behavior, profitability, and future expansion.

---

## 🎯 Business Objectives

- Analyze charging demand and station utilization.
- Understand customer charging behavior.
- Evaluate operational efficiency across charging stations.
- Measure financial performance using ROI and Payback Period.
- Identify high-potential districts for future expansion.

# 🧹 Data Preparation

The raw datasets were cleaned and transformed using **Power Query** before analysis.

Cleaning steps included:

- Removed duplicate records
- Handled missing values
- Standardized data types
- Corrected date and time formats
- Created calculated columns
- Built relationships using Power Pivot Data Model
- Validated financial and operational data
